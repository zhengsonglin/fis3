#### 一、安装fis3（确保已经安装了node)
执行cmd命令:  `npm install -g fis3`
</br>
安装完成后，测试是否安装成功，输入命令：`fis3 -v`
</br>
</br>
</br>
#### 二、打包步骤
##### 浏览器自动刷新(两种方法选一种即可，推荐第二种)
第一种：运行命令  `fis3 release -wL`
</br>
第二种：保存该文件，以后修改文件之前先双击dev.bat文件，每次修改后保存文件，该程序自动记录修改时间，并启动网页即时刷新。请勿关闭该窗口 !!!
